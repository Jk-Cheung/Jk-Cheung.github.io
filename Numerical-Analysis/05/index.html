本节主要介绍了 分段低次插值法.

点击查看索引

一般地, 我们总是认为, 插值多项式的次数越高其逼近程度越好, 但是事实并非如此. 20 世纪初, Runge 就给出了一个当 $n \to \infty$ 时, 插值多项式在区间段点处出现不收敛的函数, 若感兴趣可查看 WikiPedia 或 百度百科.
为了防止 Runge 现象的发生, 我们可以使用下面介绍的 分段低次插值.

分段线性插值

分段线性插值就是通过插值点用折线段连接起来逼近 $f(x)$, 设已知节点 $a = x_0 < x_1 < \cdots, < x_n = b$ 上的函数值 $f_0, f_1, \cdots, f_n$, 记 $h_k = x_{k + 1} - x_k, h = \max \limits_k{h_k}$, 求一折线函数 $I_h(x)$ 满足:

$$ \begin{cases} I_h(x) \in C[a, b]; \\ I_h(x_k) = f_k(k = 0, 1, \cdots, n); \\ I_h(x) \quad linear, x \in [x_k, x_{k + 1}](k = 0, 1, \cdots, n) \end{cases} $$

则称 $I_h(x)$ 为分段线性插值函数.
由定义可知 $I_h(x)$ 在每个小区间 $[x_k, x_{k + 1}]$ 上可表示为:

$$ I_h(x) = \frac{x - x_{k + 1}}{x_k - x_{k + 1}}f_k + \frac{x - x_K}{x_{k + 1} - x_k}f_{k + 1}, x_k \leq x \leq x_{k + 1}, k = 0, 1, \cdots, n - 1. $$

分段插值的误差估计可利用插值余项得到:

$$ \max_{x_k \leq x \leq x_{k + 1}}|f(x) - I_h(x)| \leq \frac{M_2}{2}\max_{x_k \leq x \leq x_{k + 1}}|(x - x_k)(x - x_{k + 1})| $$

或:

$$ \max_{x_k \leq x \leq x_{k + 1}}|f(x) - I_h(x)| \leq \frac{M_2}{8}h^2 $$

其中 $M_2 = \max\limits_{a \leq x \leq b}|f’’(x)|$

由此可知, $\lim\limits_{h \to 0}I_h(x) = f(x)$ 在 $[a, b]$ 上一致成立, 故 $I_h(x)$ 在 $[a, b]$ 上一致收敛到 $f(x)$.

分段三次 Hermite 插值

分段线性插值函数的导数是间断的, 若在节点 $x_k(k = 0, 1, \cdots, n)$ 上除了已知函数值 $f_k$ 外还给出导数值 $f’_k = m_k(k = 0, 1, \cdots, n)$, 这样就可以构造一个导数连续的分段插值函数 $I_h(x)$, 它满足:

$$ \begin{cases} I_h(x) \in C^1[a, b]; \\ I_h(x_k) = f_k, I’_h(x_k) = f’_k(k = 0, 1, \cdots, n) \\ \partial(I_h(x)) = 3, x \in [x_k, x_{k + 1}](k = 1, 2, \cdots, n) \end{cases} $$

根据两点三次 Hermite 插值多项式可知, $I_h$ 在区间 $[x_k, x_{k + 1}]$ 上的表达式为:

$$ \begin{aligned} I_h(x) = &\left(1 + 2 \dfrac{x - x_k}{x_{k + 1} - x_k}\right)\left(\dfrac{x - x_{k + 1}}{x_k - x_{k + 1}}\right)^2 f_k \\ &+ \left(1 + 2 \dfrac{x - x_{k + 1}}{x_k - x_{k + 1}}\right)\left(\dfrac{x - x_k}{x_{k + 1} - x_k}\right)^2 f_{k + 1} \\ &+ (x - x_k) \left(\dfrac{x - x_{k + 1}}{x_k - x_{k + 1}}\right)^2 y’k \\ &+ (x - x{k + 1}) \left(\dfrac{x - x_k}{x_{k + 1} - x_k}\right)^2 y’_{k + 1} \end{aligned} $$

上式对于 $k = 0, 1, \cdots, n - 1$ 成立. 对其进行误差估计, 可得:

$$ |f(x) - I_h(x)| \leq \frac{1}{384}h_k^4\max_{x_k \leq x \leq x_{k + 1}}|f^{(4)}(x)|, x \in [x_k, x_{k + 1}], h_k = x_{k + 1} - x_k $$

虽然分段三次 Hermite 插值比起分段线性插值效果明显改善, 但是这种插值要求给出节点上的导数值, 且只有一阶导数连续. 为改进这些缺点, 我们将在下一节中引入三次样条插值.
