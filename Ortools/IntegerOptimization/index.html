使用 Ortools 解决整数优化问题.

导入模块

通过以下代码从ortools.linear_solver导入pywraplp：

from ortools.linear_solver import pywraplp

定义数据函数

def create_data_model():
    data = {}
    data['cons_coeffs'] = [['cons_1'], 
                           ['cons_2'], 
                           ... , 
                           ['cons_n']]
    data['bounds'] = [['sup'],
                      ['inf']]
    data['obj_coeffs']
    data['var_bounds']
    data['num_vars']
    data['num_cons']
    return data

def create_data_model():
    data = {}
    data['cons_coeffs'] = [[a11, a12, a13], 
                           [a21, a22, a23]]
    data['bounds'] = [[b1, b2],
                      [c1, c2]]
    data['obj_coeffs'] = [k1, k2, k3]
    data['var_bounds'] = [[i1, i2, i3],
                          [s1, s2, s3]]
    data['num_vars'] = 3
    data['num_cons'] = 2
    return data

声明优化器

通过以下代码声明一个优化器实体：

solver = pywraplp.Solver.CreateSolver('SCIP')

其中，‘SCIP’ 为整数优化常用求解器

读取数据并创建变量与限制条件

读取数据：

data = creat_data_model()

创建变量：

x = {}
for j in range(data['num_vars']):
    x[j] = solver.IntVar(data['var_bounds'][0][j], 
                         data['var_bounds'][1][j],
                         'x[%i]' % j)

创建限制条件：

for j in range(data['num_cons']):
    constraint = solver.RowConstraint(data['bounds'][0][j],
                                      data['bounds'][1][j],
                                      'cons[%i] %j')
    for k in range(data['num_vars']):
        constraint.SetCoefficient(x[k], data['cons_coeffs'][j][k])

其中，方法IntVar()与方法RowConstraint()中的参数为(lower bounds, upper bounds, name)，其中name为非必须参数。

声明目标函数并指定优化方向

objective = solver.Objective()
for j in range(data['num_vars']):
    objectice.SetCoefficient(x[j], data['obj_coeffis'][j])
# 设定优化目标：最大值
objective.SetMaximization()
# 若优化目标为最小值，则删去上一行，并去掉下一行的‘#’
# objective.SetMinimization()

执行优化

status = solver.Solve()

输出结果

if status == pywraplp.Solver.OPTIMAL:
        print('Objective value:', solver.Objective().Value())
        for j in range(data['num_vars']):
            print(x[j].name(), '=', x[j].solution_value())
    else:
        print('The problem does not have a optimal solution.')
