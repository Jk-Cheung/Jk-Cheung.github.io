线性优化(Linear Optimization)

2022-01-16- 2022-02-10
- -
学习笔记-数学建模ortoolsPython编程

使用 Ortools 解决线性优化问题.

导入模块

通过以下代码从ortools.linear_solver导入pywraplp：

from ortools.linear_solver import pywraplp

声明优化器

通过以下代码声明一个优化器实体：

solver = pywraplp.Solver('SolverName', pywraplp.Solver.GLOP_LINEAR_PROGRAMMING)

其中，参数 ‘SolverName’ 为优化器名，GLOP_LINEAR_PROGRAMMING为线性优化常用求解器

创建变量

通过以下代码生成变量并声明其定义域：

x = solver.NumVar(a, b, 'x')
y = solver.NumVar(c, d, 'y')

其中，参数a, b, c, d规定其定义域，而参数’x’, ‘y’规定变量名
a, b, c, d可取无穷大，正无穷与负无穷分别表示为：

# 正无穷
solver.infinity()
# 负无穷
-solver.infinity()

更为常用的方式为在代码起始声明一个变量并将正无穷赋值于它：

infi = solver.infinity()
# 正无穷
infi
# 负无穷
-infi

声明限制条件

通过以下代码声明一个限制条件：

cons1 = solver.Constraint(inf, sup)
cons1.SetCoefficient(x, a)
cons1.SetCoefficient(y, b)

其中，solver.Constraint(inf, sup)代表该限制条件的上下界；cons1.SetCoefficient(v, c)代表设置变量v在限制条件中的系数为c

故上述代码表示限制条件：$inf \leq ax+by \leq sup$

声明目标函数

通过以下代码声明目标函数：

objective = solver.Objective()
objective.SetCoefficient(x, a)
objective.SetCoefficient(y, b)

其中，SetCoefficient() 的作用与限制条件中相同，故上述代码表示目标函数：

指定优化方向

通过以下代码指定求目标函数在限制条件中的最大值还是最小值：

# 求最大值
objective.SetMaximization()
# 求最小值
objective.SetMinimization()

执行优化

通过以下代码进行优化：

solver.Solve()

输出参数与结果

# 变量数量
solver.NumVariables()
# 限制条件数量
solver.NumConstraints()
# 优化结果
x.solution_value()
y.solution_value()
